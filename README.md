# GREENPOINT RUNNERS

The GPR landing page is the front-end of the Greenpoint Runners digital presence hosted at [greenpointrunners.com](https://greenpointrunners.com). Built bu BanditLabs.

## Quickstart

### Install dependencies

```sh
yarn
```

### Run dev

```sh
yarn dev
```

### Build all packages

```sh
yarn build
```

### Run Linter

```sh
yarn lint
```

### Run Prettier

```sh
yarn format
```
